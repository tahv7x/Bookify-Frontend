## Description
React + TypeScript frontend for **Bookify project** using Vite.  
Clean, modular, production-ready structure with ESLint and best practices.

## Features
- React + TypeScript + Vite
- HMR (Hot Module Replacement)
- Environment variables support (`.env`)
- Modular folder structure
- Axios configured for API calls
- ESLint configured for type-safe and React-specific rules

## Installation
```bash
git clone https://github.com/USERNAME/bookify-frontend.git
cd bookify-frontend
